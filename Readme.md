# Project: SKNEET-LLM-Core-Archive
## Overview
This repository contains pre-alpha training datasets for the SKNEET Large Language Model. 
Due to the 2GB per-file limit on GitHub Releases, the raw binary shards are partitioned using 7-zip (LZMA2).

## Access Policy
The dataset is encrypted to protect intellectual property. 
For decryption keys, please refer to miu2846 for authorization.

## Reassembly

To restore the shards: `7z x miu2846_SKNEET_AIO_Max2026.2.10.7z.001`
